# Editor logos

These assets are bundled so editor buttons work offline and do not depend on
macOS returning an application icon at runtime.

- `vscode.svg`: [Iconify Logos collection](https://icon-sets.iconify.design/logos/visual-studio-code/)
- `cursor.svg`: [Simple Icons — Cursor](https://simpleicons.org/?q=cursor)
- `xcode.svg`: [Simple Icons — Xcode](https://simpleicons.org/?q=xcode)
